Box Model

  • Content: A área de conteúdo onde o texto e as imagens aparecem.
  • Padding: O espaço ao redor do conteúdo.
  • Border: A borda ao redor do padding.
  • Margin: O espaço ao redor da borda.

Layout

  • Flexbox: Para layouts flexíveis e responsivos.
  • Grid: Para criar layouts de grade bidimensionais.
  • Positioning: Para posicionar elementos de forma absoluta, relativa, fixa ou sticky.
